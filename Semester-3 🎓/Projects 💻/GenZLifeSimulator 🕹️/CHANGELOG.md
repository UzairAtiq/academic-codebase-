# Changelog

All notable changes to the GenZ Life Simulator project will be documented in this file.

## [1.0.0] - 2025-11-23

### Added
- Initial release of GenZ Life Simulator chatbot
- Interactive chat functionality with 30+ Gen Z themed responses
- Life simulation game with 4 main activities:
  - Study (affects energy and points)
  - Eat (restores energy)
  - Play Games (gaming experience)
  - Sleep (full energy restore)
- Stats tracking system (Energy, Points, Mood)
- Anime quotes feature using external API
- User greeting and information collection
- Comprehensive README.md documentation
- Version control explanation document
- .gitignore for clean repository management

### Features
- Real-time stat updates based on activities
- Dynamic mood indicators with emojis
- External API integration for anime quotes
- Gen Z slang and cultural references throughout
- User-friendly terminal interface

### Technical
- Built with Python in Jupyter Notebook
- Uses requests library for API calls
- Random module for quote selection
- Dictionary-based response system
